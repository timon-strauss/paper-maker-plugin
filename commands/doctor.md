# Prerequisites

1. **Typst:** This Plugin uses Typst so it must be installed. Search for typst installation on the maschine using `typst --version`. If not there ask the user to download it. Suggestion for download from `brew`.
2. **Typst Template:** For any assignment using a template is recommended. If there is none suggest to use one for the assignment. The DHBW Template for typst can be optained at: `https://typst.app/universe/package/clean-dhbw/`
3. **NotebookLM MCP:** MCP for using NotebookLM for handling sources. If the MCP can not be connected suggest to download the local version at: `https://github.com/jacob-bd/notebooklm-mcp-cli` or if already done to start it. Check also for authentication by using `nlm notebook list` to see if you can view them all. If not ask me to authenticate using `nlm login`
